¿Por qué has elegido composición o agregación entre Vehiculo y Reparacion?

 He elegido la union fuerte ya que si vehiculo desaparece reparaciones desaparece junto a el

¿Qué ventaja tiene usar la interfaz Especialista?

Las ventajas principales de la interfaz son que te deja un codigo es escalable, limpio y 
reutilizable. La utilidad que tiene una interfaz es bastante buena para usar sus metodos
en el resto de clases sobreescribiendo estos mismos.
